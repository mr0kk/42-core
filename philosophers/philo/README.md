# Philosophers

## Description
Philosophers is a project that dives into the concurrent programming and process threading. The task was to solve synchronization problem: a group of philosophers sits at a round table with a bowl of spaghetti in the middle. 
The life of a philosopher is looks like: they eat, sleep, and think. But to eat, they must hold two forks. Since there are only as many forks as there are philosophers, they must compete for resources without communicating with each other. 
Each philosopher in simulation is represented as a separate thread.
